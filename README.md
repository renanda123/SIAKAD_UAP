# SIAKAD_UAP

SIAKAD_UAP adalah aplikasi Sistem Informasi Akademik berbasis Android yang dikembangkan menggunakan Java di Android Studio. Aplikasi ini dibuat sebagai media pembelajaran dan penyelesaian tugas Mobile Programming dengan mensimulasikan layanan akademik mahasiswa Universitas Aisyah Pringsewu.

## Deskripsi

Aplikasi menyediakan berbagai fitur akademik yang umum digunakan oleh mahasiswa, seperti melihat profil, jadwal kuliah, KRS, KHS, transkrip nilai, informasi akademik, hingga pembayaran UKT. Seluruh data yang digunakan merupakan data dummy untuk keperluan demonstrasi.

## Fitur

- Login
- Dashboard
- Profil Mahasiswa
- Biodata Mahasiswa
- Kartu Rencana Studi (KRS)
- Kartu Hasil Studi (KHS)
- Jadwal Perkuliahan
- Jadwal Ujian
- Transkrip Nilai
- Informasi Akademik
- Pengumuman
- Kalender Akademik
- Pembayaran UKT
- Riwayat Pembayaran
- Notifikasi
- Pengaturan
- Tentang Aplikasi
- Logout

## Cara Menjalankan

1. Clone repository.

```bash
git clone https://github.com/renanda123/SIAKAD_UAP.git
```

2. Buka project menggunakan Android Studio.

3. Tunggu proses Gradle Sync hingga selesai.

4. Jalankan aplikasi menggunakan emulator atau perangkat Android.

## Persyaratan

- Android Studio Hedgehog atau versi yang lebih baru
- JDK 17
- Android SDK 21 atau lebih tinggi
- Gradle sesuai konfigurasi project

Project ini dikembangkan sebagai implementasi pembelajaran pengembangan aplikasi Android menggunakan Java serta sebagai simulasi sistem informasi akademik untuk lingkungan perguruan tinggi.
